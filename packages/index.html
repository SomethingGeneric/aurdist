aurdist - AUR Package Repository

This is a pre-built repository of AUR (Arch User Repository) packages.

Usage

To use this repository with pacman, add the following to your /etc/pacman.conf:

[aurdist]
SigLevel = Never
Server = https://aur.mattcompton.dev/

Then update your package database:

sudo pacman -Sy

You can now install packages from this repository:

sudo pacman -S package-name
Note: This repository is automatically updated when new package versions are available. The packages are built in an official Arch Linux container environment for compatibility.

Available Files

This repository contains:

  • *.pkg.tar.zst - Built package files
  • aurdist.db.tar.zst - Repository database
  • aurdist.files.tar.zst - File list database
